플러터란?

Flutter는 고성능, 고품질의 iOS, Android 앱과 웹을 단일 코드 베이스로 개발할 수 있는 구글의 모바일 UI 프레임워크입니다

단일 코드 베이스로 개발 가능 (크로스 플랫폼)

플러터의 거의 모든 것은 위젯이기 때문에 위젯에 대해서만 개념을 숙지하면 쉽게 개발 가능합니다.

Stateless Widget vs Stateful Widget
(Assetimage, Text) (Scrollable, Animatable)

AOT (Ahead Of Time) : 사전 컴파일을 해서 처리한다
JIT (Just In Time) : 런타임에서 실행할 수 있는 중간언어로 변환한 후 다트의 가상머신을 통해서 중간언어를 실시간으로 번역합니다
		AOT에 비해 실행이 느리다

타입 추론

Dart 언어는 타입 추론을 지원합니다. 값이 들어올 때 타입을 추론하여 변수를 초기화합니다. (var, dynamic)

var n1 = 1;

var d1 = 10.1;

dynamic n1 = 1;


산술연산자
사칙연산, dart는 몫을 구하는 연산자 포함되어 있다

비교연산자
두 개의 값을 비교하여 결과를 참/거짓으로 반환하는 연산자입니다
(==), (!=), (<), (>) ...

논리연산자
두 개의 참/거짓 값으로 새로운 참/거짓을 반환하는 연산자입니다
(&&), (||), (!)

조건문 - 조건이 참이면 참, 아니면 거짓

삼항연산자

조건식? 참이면 실행 : 거짓이면 실행

null 대체 연산자

변수의 값이 null 이면 ?? 뒤의 값이 출력되고, null이 아니면 변수의 값이 출력됩니다

var username = null;

username ?? "홍길동"; --> 홍길동
username = "이순신";
username ?? "홍길동"; --> 이순신
