Mixin

Mixin은 여러 클래스 계층에서 클래스의 코드를 재사용하는 방법입니다. Mixin을 사용하게 되면 다중상속 문제를 해결할 수 있고

컴포지션을 사용하지 않고 다른 클래스의 코드를 재사용할 수 있습니다.


List

List는 데이터의 중복이 가능하고 순서가 있는 자료를 담는 컬렉션입니다. 자료는 순차적으로 index를 생성하여 쌓이게 됩니다.

Map

Map은 키과 값의 쌍으로 이루어진 컬렉션입니다. List는 index 번호로 값을 찾지만 Map은 키로 값을 찾아냅니다.

Set

집합을 표현하는 컬렉션입니다. 데이터의 중복이 허용되지 않기 때문에 로또 번호 생성기등을 만들 때 유용하게 사용할 수 있습니다.


Map 함수

반복되는 값을 하나씩 변형하기 위해 사용합니다.

Where 함수

조건을 필터링할 때 사용하기 때문에 컬렉션에 담긴 데이터를 삭제할 때 많이 사용합니다
